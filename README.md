# Universidad
Este repositorio contiene las tareas universitarias que tienen que hacer con lo universitario. Ojalá que dejándolas aquí pueda observarse cómo voy aumentando mi capabilidad programática a través de mi carrera.

No prohibo que se utilicen los programas publicados aquí, pero aseguro que en otros repositorios hay programas de mejor calidad. Por eso, sugiero que se usen aquellos programas, los que hacen los demás, y se vuelva aquí si se piensa que usar los míos es más preferible o risible.

Eso fue todo lo que quería decir. Disfruten.
